# Code of Conduct

Router Universal is built for thoughtful, high-trust collaboration.

Be respectful, specific, and evidence-driven. Critique ideas and code rather than people. Do not
harass, threaten, discriminate, expose private information, or deliberately degrade another
contributor's work. Security concerns should be reported privately through the process in
`SECURITY.md`.

Maintainers may edit or remove contributions and participation that violate these expectations.
Serious or repeated violations may result in a permanent ban from project spaces.
