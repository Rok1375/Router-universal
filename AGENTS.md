# Instructions for AI coding agents

Read these files before editing:

1. `START_HERE.md`
2. `docs/constitution.md`
3. `docs/project-state.md`
4. `docs/architecture.md`
5. The relevant prompt in `prompts/milestones/`

Required loop: understand, inspect, classify, discover, plan, implement, verify, critique, document.
Use the smallest relevant context. Reuse existing contracts before creating abstractions. Never
claim success without running the strongest available validation. Never silently grant permissions,
execute destructive actions, expose secrets, or let the system rewrite itself without human review.
