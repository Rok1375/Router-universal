# Roadmap

## Foundation — complete

Typed contracts, deterministic analyzer, capability registry, selection, planning, context optimizer,
policy engine, checkpoint store, event bus, adapter SDK, API, CLI, tests, CI, docs, and prompts.

## Integration wave

- Dedicated MCP client adapter and server discovery.
- OpenClaw skill and client adapter.
- OpenHands / OpenCode / Hermes coding-agent adapters.
- VS Code extension bridge and progress stream.
- Capability catalog importer for existing local skill directories.

## Intelligence wave

- Pluggable structured-output LLM analyzer.
- Semantic capability retrieval with local embeddings.
- Plan repair and bounded retry policies.
- Independent verifier routing.
- Evidence-based reusable skill proposals.

## Experience wave

- Desktop tray client, voice wake flow, plan preview, permission controls, and activity timeline.
- Multi-device hybrid mode with a cloud control plane and local execution nodes.
